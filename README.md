# Aplikasi Keuangan UMKM - User Flow Documentation

Halaman web statis standalone yang menampilkan seluruh User Flow fungsional dari perancangan Aplikasi Keuangan UMKM.

## Teknologi
- HTML5 + CSS3 + Vanilla JavaScript (tanpa framework)
- Responsive Design
- Modern CSS (flexbox, grid, animations)

## Cara Menggunakan
Cukup buka file `index.html` di browser — tidak perlu server.
